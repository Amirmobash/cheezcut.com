# CheezCut Website Package

## Deutsch

Dieses Paket enthält eine **komplette statische Website in Englisch und Deutsch** für **CheezCut**.

### Projektpositionierung
CheezCut wird als **inventor-led concept** von **Amir Mobasheraghdam** dargestellt.
Das Konzept ist **nicht** zum Schneiden von Käse gedacht und **nicht** für allgemeine Kunststoffverpackungen.
Der Fokus liegt ausschließlich auf der **roten äußeren Wachsschicht rund um Gouda**.

### Was enthalten ist
- Englische Hauptseiten im Root-Verzeichnis
- Deutsche Seiten im Ordner `de/`
- Gemeinsame Assets in `static/`
- Die drei gelieferten Bilder direkt in `static/images/`

### Bilder in der Website
Verwendete Dateien:
- `static/images/cheezcut-hero-light.png`
- `static/images/cheezcut-render-transparent.png`
- `static/images/cheezcut-concept-layered.png`

### Bildrichtung / Ausschlüsse
Die Bildsprache wurde so ausgerichtet, dass typische Fehlinterpretationen vermieden werden:
- sliced cheese
- sandwich cheese
- cheese layers with red coating
- pizza cheese
- shredded cheese
- plastic packaging
- wrapper bag
- cartoon
- blurry
- unrealistic tool
- hands
- knife
- messy kitchen
- dark background
- toy-like object
- distorted cheese
- text
- watermark
- logo overlay

### Deployment auf Render
- Service-Typ: **Static Site**
- Publish Directory: `./`
- Build Command: leer lassen
- Environment Variables: leer lassen

### Vor Livegang anpassen
- `hello@cheezcut.com` durch echte Kontaktdaten ersetzen
- Datenschutz / Nutzungsbedingungen juristisch finalisieren
- Bei Bedarf ein echtes Kontaktformular anschließen

## English

This package contains a **complete static website in English and German** for **CheezCut**.

### Project positioning
CheezCut is presented as an **inventor-led concept** by **Amir Mobasheraghdam**.
It is **not** intended for slicing cheese and **not** intended for ordinary plastic packaging.
Its focus is the **red outer wax rind around Gouda cheese**.

### What is included
- English main pages in the root directory
- German pages inside `de/`
- Shared assets in `static/`
- The three supplied images placed directly in `static/images/`

### Images used in the website
Files included:
- `static/images/cheezcut-hero-light.png`
- `static/images/cheezcut-render-transparent.png`
- `static/images/cheezcut-concept-layered.png`

### Deployment on Render
- Service type: **Static Site**
- Publish Directory: `./`
- Build Command: leave blank
- Environment Variables: leave blank

### Before going live
- Replace `hello@cheezcut.com` with your real contact details
- Finalize privacy / terms with legal wording if needed
- Connect a real contact form backend later if desired
